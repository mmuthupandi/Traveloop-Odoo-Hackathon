## ✨ Features

- **Split-screen responsive layout**
  - Left: Login form with warm beige background
  - Right: Travel-themed banner with inspirational quote
  
- **Form Features:**
  - Email input with validation
  - Password input with show/hide toggle
  - Remember me checkbox
  - Forgot password link (UI only)
  - Error message display for invalid credentials
  
- **Design Elements:**
  - Warm beige (#F7F4EE) background
  - Forest green (#2C5F4F) buttons
  - Orange (#D97706) accent colors
  - Smooth hover effects and transitions
  - Travel-themed icons and illustrations
  
- **Responsive Design:**
  - Mobile-friendly (form only on small screens)
  - Split-screen on desktop (lg breakpoint and above)
  
## 🎨 Color Palette

- Background: `#F7F4EE` (Warm Beige)
- Primary Button: `#2C5F4F` (Forest Green)
- Accent: `#D97706` (Amber/Orange)
- Text: Gray scale

## 📱 Mobile Responsive

On mobile devices (< 1024px), the right banner is hidden and only the login form is displayed in a centered, full-width layout.
